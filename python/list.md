## List

### Create a list

#### Default initialization

```python
n = 5; # number of items
num = [0] * n
```

### With fixed initial values

```python
n = 5; # number of items
num = [-1] * n
```

### Sort in ascending order

```python
num = [5, 4, 3, 2]
num.sort() # in place
```

### sort in descending order
```python
num = [5, 4, 3, 2]
num.sort(reverse=True)
```
