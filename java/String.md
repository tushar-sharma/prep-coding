## String


| **Operations** | Description |
| --- | --- |
| Find length of string | s.length() |
| Get character at location i | s.charAt(i) |
| Get substring from i to j  | s.substring(i, j + 1) | 
| Is empty |  s.isEmpty()  |


### Iterate a string

```java
String s = "some string";
for (char c : s.toCharArray()) {
}
```
