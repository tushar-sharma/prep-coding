# Complexity

## Stack

| Operations | Big Oh|
|--|--|
|Access | O(n) |
|Search | O(n) |
|Insert | O(1) |
|Remove | O(1) |

## Binary Search Tree

| Opertions| Big Oh |
|--|--|
| Insert | O(h)  |
| Min | O(h) |
| Delete | O(h) |
| Rank | O(h) |


where h is height of tree. Best case it is lg(n) or n for worst case.
